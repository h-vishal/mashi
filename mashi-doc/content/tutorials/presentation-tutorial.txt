#Mashi Presentation Tutorial

Mashi provids a way to make presentations based on web standards. Each slide is represented as a DIV elements in XHTML/XML notation. You can nest XHTML elemente like P, EM, STRONG, Ul, Ol, LI, TABLE inside these DIV elements. The only limitation is using nested DIV elements (DIVs inside a silde).

######HTML

	// Head
	<script type="text/javascript" 
	  src="../system/javascripts/mashi.js"></script>
		
	// Body
	<div id="splash">
		<div id="mySlideshow" class="mashi"></div>
	</div>
	


######XML

	<!-- my-hello-world-presentation.xml -->
	<?xml version="1.0" encoding="UTF-8"?>
	<html>
	  <head>
	    <title>Hello world</title>
	  </head>
	  <body>

	    <!-- Slide 1 -->
	    <div>
		  <h2>Hello world program</h2>
		  <p>
		    "Hello World" program is a computer program which prints 
		     out "Hello World!" on a display device. It is used in many 
		     introductory tutorials for teaching a programming language. 
		  </p>
		  <p>
		    Such a program is typically one of the simplest programs 
		    possible in most computer languages. It is often considered 
		    to be tradition among programmers for people attempting to 
		    learn a new programming language to write a "Hello World!" 
		    program as one of the first steps of learning that particular 
		    language.
		  </p>
	    </div>
	    <!-- END Slide 1 -->

	    <!-- Slide 2 -->
	    <div>		
	      <h3>Purpose</h3>
	      <p>
	        A "hello world" program has become the traditional first 
	        program that many people learn. In general, it is simple 
	        enough that people who have no previous experience with 
	        computer programming can easily understand it, especially 
	        with the guidance of a teacher or a written guide.
	      </p>
	    </div>
	    <!-- END Slide 2 -->

	  </body>
	</html>	




######CSS

Write the styles in the 'custom.css' file inside the 'application' directory.

	#splash {
	  width: 1024px;
	  height: 512px;
	}
	
######Javascript
	
	var mySlideshow = new mashi.application();

	/* modules */
	mashi.module('mashi.preloader');
	mashi.module('mashi.controls');
	mashi.module('mashi.slideshow.create');


	mashi.slideshow.create({
	  source: 'my-hello-world-presentation.xml',
	  bgimage: 'images/hello-world-background.jpg'
	});

	// Add a frame at the end 
	mySlideshow.add(100, function() {
	  mySlideshow.set({ 
	    html: 
	      '<span style="text-align:center">' +
	      '  <em>Automatically generated with the</em><br/>' +
	      '   Mashi slideshow converter' + 
	      '</span>' +
	      '<span style="text-align:center">' +
	      '  <em>author</em>' +
	      '  <a alt="" href="http://vankoch.nl">Vincent van Koch</a>' +
	      '  <em>)</em>' +
	      '</span>'
	  }); 
	});

	window.onload = function() {
	    mySlideshow.config({
	        meta: {
	            namespace: 'mySlideshow',
	            title: 'My Hello World Presentation'
	        },
	        animation: {
	            start: true
	        },
	        buttons: {
	            type: 'slideshow'
	        } 
	    });
	};
