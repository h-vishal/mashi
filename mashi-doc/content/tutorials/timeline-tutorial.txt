#Mashi Timeline Tutorial

'mashi.timeline' is the base function of Mashi. With this you can start timeline-controlled JavaScript functions. You can change the style of text depending on time, for example.

The following mini example shows how to let an alert appear after 10 seconds and let it disappear after 1 minute.

	var myTimeline = new mashi.timeline();  
	
	myTimeline.add(10000, function() {
	  document.getElementById('notice').style.visibility = 'visible';
	});
	
	myTimeline.add(60000, function() {
	  document.getElementById('notice').style.visibility = 'hidden';
	});
	
	document.onload = function() {
	  myTimeline.start();
	}
