#Mashi Application Tutorial

'mashi.application' is the main function of mashi. Like with 'mashi.timeline' you can start any JavaScript actions. Beyond that Mashi provides many methods to easily create objects on the stage and to move and manipulate them.

The following example shows an ad banner preloading images and looping the app endlessly. Clicking the banner targets a landing page.

######HTML

	// Head
	<script type="text/javascript">
	  // Pfad zum 'application'-Ordner
	  // Muss nicht angegeben werden, wenn sich das HTML-Dokument 
	  // selbst im 'application'-Ordner der Mashi App befindet.
	  var MASHI_APP_PATH = 'myapps/advertising/myapp'; 
	</script>
	<script type="text/javascript" 
	  src="../system/javascripts/mashi.js"></script>
		
	// Body
	<div id="splash">
		<div id="myApp" class="mashi"></div>
	</div>

######CSS

Put the styles in 'custom.css' inside the 'application' directory.

	#splash {
	  width: 1024px;
	  height: 512px;
	  cursor: 'pointer';
	}
	
######Javascript

	var myApp = new mashi.application();  

	/* load modules */
	mashi.module('mashi.preloader');
	mashi.module('mashi.object.animation');

	var TEXT;

	myApp.add(3000, function() {

	  myApp.set({
	    style: {
	      backgroundImage: "url(" + MASHI_APP_PATH + "content/images/background.jpg)"
	    }
	  });

	  TEXT = new mashi.object({
	    id: "text_layer",
	    style: {
	        width: "130px",
	        height: "50px",
	        left: "0px",
	        top: "-800px"
	    }
	  });

	  TEXT.set({
	    html: 'Hello World!',
	    style: {
	      backgroundImage: "url(" + MASHI_APP_PATH + "content/images/trans_white.png)",
	      textAlign: "center",
	      color: "black",
	      padding: "15px",
	      fontSize: "16px"
	    }
	  });

	  TEXT.moveTo({
	      left: "20px",
	      top: "120px"
	  });

	});

	myApp.add(3000, function() {

	  TEXT.set({
	    html: 'Hey! Nobody at home?',
	    style: {
	      fontSize: "43px"
	    }
	  });

	});

	myApp.add(3000, function() {

	  TEXT.moveTo({
	      left: "0px",
	      top: "-500px"
	  });

	});
	
	window.onload = function() {
	
	  myApp.config({
	      meta: {
	      	namespace: 'app', 
	        title: 'Hello World!',
	        author: 'Advertising Inc.'	
	      },
	      animation : {
	          start: true,
	          rewind: true
	      }
	  });
	
	  var splash = document.getElementById('splash');
	  splash.title = 'read more ..';  
	  splash.onclick = function() {
	    location.href = 'http://www.advertising-inc.com/product?v=PsekY55zhzo';
	  }
	
	};
