#Core Reference - mashi.fetch

Loading Content from external file via XHR/XPath

#####Syntax

	var content = mashi.fetch(
	  XMLSOURCE, // External document (url-path)
	  XPATH // Node via XPath expression
	);

#####Example

	app.set({
  	  html: mashi.fetch(
	    appPath + 'content/hello.lang.xml',
	    '/html/body/div[@id="hello"]/div[@lang="' + language + '"]' 
	  )
	});
