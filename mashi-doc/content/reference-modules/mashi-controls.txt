#Module Reference - mashi.controls

The module 'Controls' provides buttons for controlling a Mashi app. Therefore it is only used in the context of 'mashi.application' not in 'mashi.timeline'!

#####Integration

	mashi.module('mashi.controls');


#####Configuration and default values

	INSTANCE.config(
	
	  meta : {
        namespace: 'MYAPP', // required - Name of the instance
      },

	  ...

      /* for module mashi.controls */
	  buttons : { 
        type: false, // optional - false/single/movie/slideshow/all
        position : {
          vertical: 'bottom', // optional - bottom/middle/top
          horizontal: 'right' // optional - right/center/left
        }
      }
	);

#####Control types

You can use following types in the configuration:

######buttons.type: false

No buttons will be displayed (default).

######buttons.type: 'single'

A toggle button with the functions 'Start' and 'Stop' will be displayed. This means 'Stop' will be displayed while the app runs and 'Start' while it stands still.

![Alt Screenshot buttons single stop](content/images/controls-single-stop.jpg "Screenshot buttons single stop")

![Alt Screenshot buttons single start](content/images/controls-single-start.jpg "Screenshot buttons single start")

######buttons.type: 'movie'

'Start' and 'Stop' buttons are displayed next to each other.

![Alt Screenshot buttons movie](content/images/controls-movie.jpg "Screenshot buttons movie")

######buttons.type: 'slideshow'

Four buttons are displayed: 'first frame', 'previous frame', 'next frame', 'last frame'.

![Alt Screenshot buttons slideshow](content/images/controls-slideshow.jpg "Screenshot buttons slideshow")

######buttons.type: 'all'

All 6 buttons are displayed: 'first frame', 'previous frame', 'start', 'stop', 'next frame', 'last frame'.

![Alt Screenshot buttons all](content/images/controls-all.jpg "Screenshot buttons all")


#####Control position

The position of the controls can be set in the configuration file:

######buttons.position.vertical

You can use 'top', 'middle' oder 'bottom' as values. 'bottom' is the default value if no value is set.

######buttons.position.horizontal

You can use 'left', 'center' or 'right' as values. 'right' is the default value if no value is set.

#####Example

	var app = new mashi.application();
	mashi.module('mashi.controls');
	
	// add some time frames
	
	window.onload = function() {
	  app.config({
	    meta: {
	      namespace: 'app'
	    },
	    buttons: {
	      type: 'single',
		  position : {
            vertical: 'top', 
            horizontal: 'left'
          }
	    },
	    animation: {
	      start: true
	    }
	  });
	};
