#Module Reference - mashi.object.typewriter

Extends the 'mashi.object' with a method that creates a typewriter effect.

#####Integration

Note: mashi.object does not have to be explicitly loaded, because the extensions automatically loads the parent module.

	mashi.module('mashi.object.typewriter');
	
#####Methods


######run()

	OBJECT.run(
      html: STRING,   // required
      delay: INTEGER, // required
      wait: INTEGER   // required
	);
	
######Example

	MyObjects.infolayer.run({
        html: '<b>Fernsehturm Berlin</b>',
        delay: 20,
        wait: 2000
    });
