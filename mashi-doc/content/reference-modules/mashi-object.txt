#Module Reference - mashi.object

Creates objects on the stage.

#####Integration

	mashi.module('mashi.object');
		
#####Usage

	OBJECT = new mashi.object({
	  id: STRING,
	  html: STRING,
	  style: {
	    // List of CSS Properties
	  }
	});
	
#####Publish your Objects

Please make sure that all instance names of objects are listet at the beginning of a frameset. Otherwise they won't be recognized by the player on runtime:

	var MyObject_first, MyObject_second;
	
	app.add(1000,
	function() {

	  MyObject_first = new mashi.object({
	    ...
	  });
	
	});
	
	app.add(1000,
	function() {

	  MyObject_second = new mashi.object({
	    ...
	  });
	
	});

<br />
You can use a namespace if you want to code more comfortable or just reduce the count of global variables:

		var MyObjects = new Object();

		app.add(1000,
		function() {

		  MyObjects.first = new mashi.object({
		    ...
		  });

		});

		app.add(1000,
		function() {

		  MyObjects.first.appear({
		    ...
		  });

		});




#####Methods

mashi.object only has one method: set. With this method you can change an object after creation.

######set()

	OBJECT.set({
	  html: STRING,
	  style: {
	    // List of CSS Properties
	  }
	});
	
#####Example:	
	
	TEXTLAYER = new mashi.object({
	  id: "textlayer",
	  style: {
	    width: "351px",
		height: "76px",
		left: "-500px",
		top: "10px",
		backgroundImage: "url(" + MASHI_APP_PATH + "content/images/textlayer_1.png)",
		padding: "15px",
		fontSize: "13px",
		lineHeight: "1.15em"
  	  }
	});
	
	TEXTLAYER.set({
        style: {
            backgroundImage: "url(" + MASHI_APP_PATH + "content/images/textlayer_2.png)"
        }
    });
    
	TEXTLAYER.set({
        html: 'Hello World!'
    });
